📰SHABBADUBAH NEWS PORTAL

Welcome to Shabbadubah News Portal, a modern and responsive news website designed to deliver the latest updates in technology, entertainment, and global affairs. This project showcases a clean layout, engaging content, and interactive features inspired by major platforms like BBC and CNN.

Live Website

 https://tsiunyanesechaba26-ops.github.io/Shabbadubah-/

Project Overview

GlobalNews Portal is a multi-page website built using HTML, CSS. It provides users with an engaging experience through structured news articles, multimedia content, and smooth navigation.

 Features
 Home Page with featured news articles
 Categories Page for browsing different news topics
 Article Pages with detailed news content
Multimedia Page with embedded videos and images
Contact Page with a functional form layout
 Back Button Navigation for better user experience
 Smooth Animations (hover effects, fade-ins, transitions)
 Responsive Design for mobile and desktop
 Technologies Used
HTML – Structure of the website
CSS – Styling and layout
JavaScript – Basic interactivity and animations
GitHub Pages – Hosting the website
